# 💊 PharmaPulse AI-Powered CRM for Healthcare Professionals

PharmaPulse AI is a full-stack AI-powered Customer Relationship Management (CRM) platform built for pharmaceutical organizations to streamline Healthcare Professional (HCP) engagement. The application combines modern web technologies with Generative AI and LangGraph to automate interaction logging, intelligent search, note summarization, and AI-assisted decision support.

Designed with scalability and usability in mind, PharmaPulse AI enables medical representatives and sales teams to efficiently manage HCP profiles, track interactions, analyze engagement, and generate AI-driven insights in real time.

# ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/rahulgupta-cse/PharmaPulse_AI.git
cd PharmaPulse_AI
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
